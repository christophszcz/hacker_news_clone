== Hacker News Clone 

This is a replica of the popular Hacker News website built using Zurb Foundation, CSS, vanilla JavaScript and of course Ruby on Rails. This app is running Rails 4.2.4 (Ruby 2.2.0p0).

{See it live}[https://christophszcz_hacker_news_clone.herokuapp.com/]

1. <tt>$ git clone git@github.com:christophszcz/hacker_news_clone.git</tt>
2. <tt>$ cd hackernews_clone</tt> 
3. <tt>$ bundle</tt> 
4. Enjoy! :)
